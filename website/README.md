# Curated implementation snapshot

This directory contains the public parts of the delivered website that are most relevant to the case study:

- Reusable page and navigation components
- Product and service route templates
- Structured company, product, service, navigation, and material status data
- The approved company logo used by the interface

It is intentionally not a standalone deployment package. Platform credentials, generated files, unrelated personal website routes, private conversations, and local environment details are excluded.

The production result is available at [stevewei.ca/lab/syans](https://stevewei.ca/lab/syans).
